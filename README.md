# Flower Pomodoro Timer 🌸

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![.NET](https://img.shields.io/badge/.NET-9.0-purple.svg)
![Platform](https://img.shields.io/badge/platform-Windows-0078d7.svg)

**Flower Pomodoro Timer** 是一款結合了「番茄鐘工作法」、「自動化視窗活動紀錄」與「系統效能監控」的生產力工具。它不僅能幫助您專注於工作，還能透過詳細的數據分析讓您了解時間都花在了哪些應用程式上。

## 🌟 核心功能

### 1. 雙重計時系統與多功能 Stopwatch
*   **番茄鐘模式**：預設 55 分鐘工作與 5 分鐘休息循環，自動切換狀態。
*   **獨立 Stopwatch 計時器**：迷你模式下提供獨立的計時器功能（00:00 格式），支援開始/暫停/重置，與番茄鐘狀態完全獨立，適合臨時計時需求。
*   **休息時段 — 多螢幕 & 虛擬桌面支援**：
    *   提醒短暫休息時段，幫助您放鬆眼睛與身體。
    *   可自訂圖片覆蓋整個螢幕，強制停工休息。
    *   **多螢幕支援**：每個連接的螢幕各顯示一張不同的隨機圖片。
    *   **虛擬桌面支援**：自動在使用者所在的虛擬桌面顯示，避免在不同桌面間分散。
    *   **圖片位置記憶**：允許使用者在 REST 時拖曳圖片位置，程式會記住每個螢幕的位置，下次進入 REST 時自動恢復。
    *   **雙擊快速關閉**：雙擊任一螢幕的圖片即關閉所有螢幕的覆蓋圖片。
*   **總累計計時**：追蹤當日啟動後的總投入時間。
*   **自動隔日分別統計**：當系統時間跨入隔日，自動將前一日的累積數據保存並重新計時，不因重啟程式而遺失。
*   **智慧偵測**：當系統進入鎖定畫面或待機時，自動計算「離開時間」，確保數據精確。

### 2. 視窗活動追蹤與分析
*   **自動偵測**：實時紀錄當前最上層使用的應用程式（Process）及其視窗標題。
*   **階層式統計**：以橫條圖顯示各程式的使用時間，支援展開查看該程式下不同視窗（如瀏覽器分頁）的具體分配。
*   **熱度排序**：自動根據使用時數進行排序，前 20 名活動一目了然。

### 3. 實時效能儀表板（可暫停以節省 CPU）
*   **硬體監控**：整合五大效能指標條，背景深色、條狀淺色，方便快速判讀：
    *   **CPU** 使用率 (%) — 每秒更新
    *   **RAM** 記憶體容量與比例 — 每秒更新
    *   **Disk** 磁碟讀寫速度 (MB/s) — 每秒更新
    *   **GPU** 使用率 (%) — 每 3 秒更新（降頻優化）
    *   **VRAM** 顯存佔用量 — 每 3 秒更新（降頻優化）
*   **暫停顯示效率**：右上角新增「II/▶」按鈕，可暫停五條橫條的更新，完全停止 PerformanceCounter 取樣，極大降低 CPU 使用率（只在需要時才顯示）。

### 4. 工作排程 (Work Schedule)
*   **自定義清單**：透過 `WorkList.txt` 定義不同時段的任務（支援特定日期或每日重複）。
*   **當前任務提示**：自動讀取並在介面上顯示當前應進行的工作項目。

### 5. 工時統計分析 (FormUsageAnalysis.cs)
*   **時長分析統計**：分析各種軟體使用時間，深度瞭解自己的工作習慣。
*   **自訂日期**：可自行設定分析內容的起訖日期。
*   **智慧排除選項**（預設勾選）：
    *   **排除 Flower Pomodoro Timer**：避免計時器本身的視窗佔用時間被計入統計。
    *   **排除 Idle/Unknown usage**：隱藏待機/未知程序的時間記錄。
    *   **排除 LockApp**：隱藏螢幕鎖定期間的時間記錄。

### 6. 極致的 UI/UX 體驗
*   **雙模式切換**：
    *   **完整模式**：提供完整的活動列表、詳細數據與效能監控。
    *   **迷你模式 (Mini View)**：縮小至右下角，重新排版為：
        *   **左欄**（齊左）：番茄時間、Start 按鈕、總時間。
        *   **右欄**：獨立 Stopwatch 方框（顯示計時 00:00、Start/暫停、重置按鈕）。
        *   **下方**：五條效能橫條（CPU/RAM/Disk/GPU/VRAM），橫跨全寬。
*   **視窗拖曳與位置記憶**：
    *   迷你模式下可用滑鼠拖曳視窗位置（在空白區域或非按鈕區域）。
    *   程式自動記住視窗位置，重新啟動時自動恢復。
*   **多樣化配色**：提供番茄紅、草地綠、天空藍、深灰色等主題，隨狀態或偏好切換。
*   **個人化調整**：支援透明度調整、最上層顯示切換（Always on Top）。

## 🚀 快速上手

### 系統需求
*   **作業系統**：Windows 10 1809 或更高版本。
*   **執行環境**：.NET 9.0 Runtime。

### 安裝與執行
1. 下載並解壓縮專案。
2. 執行 `FlowerPomodoroTimer.exe`。
3. 程式會自動在目錄下產生 `WorkList.txt`。

### 設定工作清單
編輯 `WorkList.txt`，格式如下：
```csv
# 格式: Day,Start,End,Task
# Day 可填寫 Monday..Sunday 或 null (代表每天)
Monday,09:00,12:00,開發新功能
null,13:30,17:30,寫程式與除錯
```

## 🛠 技術架構
*   **開發語言**：C#
*   **框架**：WinForms (.NET 9)
*   **監控技術**：
    *   `System.Diagnostics.PerformanceCounter` (硬體數據，支援選擇性停用以降低 CPU)
    *   `User32.dll` (視窗偵測 & 螢幕 DPI 感知)
    *   `Kernel32.dll` (記憶體狀態)
    *   `Registry` (顯示卡 VRAM 容量讀取)
*   **自定義控制項**：`BarChartBox` 負責高效能的繪圖渲染。
*   **性能優化**：
    *   **GPU/VRAM 降頻**：改為 3 秒更新一次（相比 CPU/RAM/Disk 的 1 秒），降低 PDH 查詢開銷。
    *   **暫停效率顯示**：右上角「II/▶」按鈕可完全停止 PerformanceCounter 取樣，節省 CPU 資源。
    *   **視窗統計降頻**：排序與重繪改為 3 秒一次，視覺差異微乎其微但 CPU 用量顯著下降。
    *   **DPI 感知縮放**：採用 `PerMonitorV2` 模式，所有尺寸自動根據螢幕 DPI 縮放，避免高 DPI 下的模糊或錯位。
