# Flower Pomodoro Timer 🌸

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![.NET](https://img.shields.io/badge/.NET-9.0-purple.svg)
![Platform](https://img.shields.io/badge/platform-Windows-0078d7.svg)

**Flower Pomodoro Timer** 是一款結合了「番茄鐘工作法」、「自動化視窗活動紀錄」與「系統效能監控」的生產力工具。它不僅能幫助您專注於工作，還能透過詳細的數據分析讓您了解時間都花在了哪些應用程式上。

## 🌟 核心功能

### 1. 雙重計時系統
*   **番茄鐘模式**：預設 55 分鐘工作與 5 分鐘休息循環，自動切換狀態。
*   **休息時段**：提醒短暫休息時段，幫助您放鬆眼睛與身體，亦可以自訂圖片覆蓋整個螢幕(多桌面與多螢幕都有效)，強制停工休息。
*   **總累計計時**：追蹤當日啟動後的總投入時間。
*   **自動隔日分別統計**：當系統時間跨入隔日，自動將前一日的累積數據保存並重新計時，不因重啟程式而遺失。
*   **智慧偵測**：當系統進入鎖定畫面或待機時，自動計算「離開時間」，確保數據精確。

### 2. 視窗活動追蹤與分析
*   **自動偵測**：實時紀錄當前最上層使用的應用程式（Process）及其視窗標題。
*   **階層式統計**：以橫條圖顯示各程式的使用時間，支援展開查看該程式下不同視窗（如瀏覽器分頁）的具體分配。
*   **熱度排序**：自動根據使用時數進行排序，前 20 名活動一目了然。

### 3. 實時效能儀表板
*   **硬體監控**：整合五大效能指標條，背景深色、條狀淺色，方便快速判讀：
    *   **CPU** 使用率 (%)
    *   **RAM** 記憶體容量與比例
    *   **Disk** 磁碟讀寫速度 (MB/s)
    *   **GPU** 使用率 (%)
    *   **VRAM** 顯存佔用量
*   **同步更新**：與秒數計時同步刷新。

### 4. 工作排程 (Work Schedule)
*   **自定義清單**：透過 `WorkList.txt` 定義不同時段的任務（支援特定日期或每日重複）。
*   **當前任務提示**：自動讀取並在介面上顯示當前應進行的工作項目。

### 5. 工時統計分析 (FormUsageAnalysis.cs)
*   **時長分析統計**：分析各種軟體使用時間，深度瞭解自己的工作習慣。
*   **自訂日期**：可自行設定分析內容的起訖日期。

### 6. 極致的 UI/UX 體驗
*   **雙模式切換**：
    *   **完整模式**：提供完整的活動列表與詳細數據。
    *   **迷你模式 (Mini View)**：縮小至右下角，僅保留計時器與效能條，減少干擾。
*   **多樣化配色**：提供番茄紅、草地綠、天空藍、深灰色等主題，隨狀態或偏好切換。
*   **個人化調整**：支援透明度調整、最上層顯示切換（Always on Top）。

## 🚀 快速上手

### 系統需求
*   **作業系統**：Windows 10 1809 或更高版本。
*   **執行環境**：.NET 9.0 Runtime。

### 安裝與執行
1. 下載並解壓縮專案。
2. 執行 `FlowerPomodoroTimer.exe`。
3. 程式會自動在目錄下產生 `WorkList.txt`。

### 設定工作清單
編輯 `WorkList.txt`，格式如下：
```csv
# 格式: Day,Start,End,Task
# Day 可填寫 Monday..Sunday 或 null (代表每天)
Monday,09:00,12:00,開發新功能
null,13:30,17:30,寫程式與除錯
```

## 🛠 技術架構
*   **開發語言**：C#
*   **框架**：WinForms (.NET 9)
*   **監控技術**：
    *   `System.Diagnostics.PerformanceCounter` (硬體數據)
    *   `User32.dll` (視窗偵測)
    *   `Kernel32.dll` (記憶體狀態)
*   **自定義控制項**：`BarChartBox` 負責高效能的繪圖渲染。
